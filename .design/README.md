# Design Assets - ConectAr HR

Esta carpeta contiene assets y mockups de diseño para el proyecto ConectAr HR.

## Contenido

- **Pantalla Login_v01.png** - Mockup de referencia para la pantalla de login
- **Logotipo_ConectAr.png** - Logo oficial de ConectAr
- **Marca_ConectAr.png** - Marca gráfica de ConectAr
- **Fondo abstracto con luces suaves.png** - Fondo utilizado en el diseño de login
- **preview.html** - Preview interactivo de los diseños

## Uso

Estos assets son para **referencia del equipo de diseño**. Los logos utilizados en el código se encuentran en:
- `public/logotipo-conectar-transparent.png`
- `public/marca-conectar-transparent.png`

## Notas

- No incluir assets de esta carpeta en el bundle de producción
- Mantener sincronizados con `public/` cuando se actualicen logos
